Picktr

Picktr is a single page photo sharing web application based off of the popular website Flickr. The stack implemented
used React and Redux JavaScript libraries for the frontend, a PostgreSQL database, and Ruby on Rails for the backend.
Image hosting was achieved through Rails' Active Storage and AWS S3.

Zombies

Zombies is a 2-D shooting game utilizing mouse and keyboard input. This project was built in a one week time period
using Vanilla JavaScript, HTML5, and CSS. Animation was done through the use of a canvas HTML element. There are
infinite waves of zombies; See how many you can survive!

Live Link || Github

Time Blocks

TimeBlocks is a is a user-friendly time management app optimized for mobile-friendly use. Visually-appealing, colored
blocks are used to represent hours of the day and customizable tasks can be dragged to fill any open time. This was a
team project implemented using the MERN stack.
